# Content Distribution

This section helps deliver the application to end users securely and with high performance.

## Content
1. **Certificate (ACM)**: Issue free SSL/TLS certificates.
2. **Application Load Balancer (ALB)**: Load balance for Backend (Spring Boot).
3. **CloudFront**: CDN to distribute Frontend (ReactJS) and Static files (S3).
4. **Route 53**: Manage Domain DNS.
