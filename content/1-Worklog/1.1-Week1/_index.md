### Week 1 Achievements:

* Understood what AWS is and grasped basic service groups: 
  * Compute
  * Storage
  * Networking 
  * Database
  * AI

* Successfully created and configured AWS Free Tier account.

* Got familiar with AWS Management Console and know how to find, access, and use services from the web interface.

* Created Budget plan, know how to track monthly costs for used services.

* Capable of connecting between web interface and CLI to manage AWS resources in parallel.

* Understood concepts of Region, Availability Zone

* Understood how resources in VPC public subnet, private subnet communicate with each other and with the external internet environment
