**In this page** you will need to introduce your worklog **how**? You complete the program in **how many weeks**? What **did you do** during those weeks?


Typically and also as a standard, a worklog is carried out in about 3 months (during the internship period) with the content of the weeks as follows:

**Week 1:** [Get familiar with AWS and basic services in AWS](1.1-week1/)

**Week 2:** [Practice using AWS services](1.2-week2/)

**Week 3:** [Design auction web database, select AWS services](1.3-week3/)

**Week 4:** [Build auction web backend, registration, login, security, account recovery features. Group meeting to choose group project topic](1.4-week4/)

**Week 5:** [Build auction web backend, system category management features. Design group project database](1.5-week5/)

**Week 6:** [Build auction web backend, auction feature. Complete backend for group project](1.6-week6/)

**Week 7:** [Build auction web system, transaction wallet management feature.](1.7-week7/)

**Week 8:** [Build auction web system, delivery feature.](1.8-week8/)

**Week 9:** [Complete backend for auction web, build project interface](1.9-week9/)

**Week 10:** [Complete home page interface, user interface for auction web.](1.10-week10/)

**Week 11:** [Complete admin, staff interface for auction web.](1.11-week11/)

**Week 12:** [Complete project, deploy to cloud environment, testing.](1.12-week12/)
